## Scene Switcher Controls

### Scene hotkeys

| Key | Result |
|-----|--------|
| `F1` | Render the Scene 0 offscreen image directly. |
| `F2` | Render the Scene 1 offscreen image directly. |
| `F3` | Render the Scene 2 target with the Scene 4 fire overlay blended on top. |
| `F4` | Run the full showcase timeline with all scene transitions and blends. |
| `F5` | Render the Scene 3 offscreen image directly. |
| `F6` | Render the Scene 4 offscreen image directly (enables live fire controls). |
